# 🧪 Software Tests — QA Board

Board kanban para gerenciamento de testes, bugs e melhorias de software.

## Funcionalidades

- **Testes** — A Testar / Testando / Testado
- **Bugs** — Reportado / Corrigindo / Corrigido
- **Melhorias** — Sugerido / Planejado / Implementado
- Dados em tempo real via Supabase Realtime
- Responsivo para mobile e desktop

## Stack

- HTML + CSS + JavaScript (vanilla)
- Supabase (banco de dados + realtime)
- Google Fonts (Permanent Marker, Bebas Neue, Space Mono)

## Rodar localmente

1. Edite `app.js` e coloque sua URL e chave do Supabase
2. Abra `index.html` no navegador, ou rode:

```bash
python3 -m http.server 3000
```

## Banco de dados

As migrações estão em `supabase/migrations/`. Rode os arquivos SQL no SQL Editor do Supabase.
